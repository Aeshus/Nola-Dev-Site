# Nola Dev's Site

- [Agile Nola](https://www.eventbrite.com/o/agile-nola-59051586693)
- [Hack Night](https://www.meetup.com/nolahacknight/)

- [#FrontEndParty](http://www.frontendparty.com/)

- A.I. Hacking In Nola
